About the Multilingual Educational Assistant

Language differences often hinder access to high-quality learning nodes. This project provides a fully localized engineering prototype that reads and dictates responses in regional Indian dialects. Designed to support rural classrooms, the codebase performs local NLP syntax checks and parses notes files offline without relying on paywalled Cloud translation servers.

Software Stack

  • Frontend: React.js Single-Page Framework
  • Backend Server: FastAPI (Python Web API framework)
  • Database Schema: SQLite + SQLAlchemy
  • Local NLP: spaCy + custom TF-IDF search
  • Voice Synthesis: gTTS stream with MD5 file cache

Project Details

This project has been developed as an academic showcase, illustrating how full-stack modular programming can optimize educational delivery models. The offline fallback handlers ensure 100% server availability and zero usage bills.
